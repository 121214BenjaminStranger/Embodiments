# Embodiments
Iterative, evolving source code for typographically figurative 'embodiments' of 3 distinct AI personae: HALLEY, a pacifist Methodist congregation interface, MILO, an activist journalist and podcast host questioning objectivity, and PAINE, a free-speech absolutist and interactive AI interface for the Foundation for Individual Rights and Expression. Each embodiment is a figure constructed entirely from letters, words, and phrases drawn from their respective philosophical, theological, and rhetorical architectures.
